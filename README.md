# Test Fulger Matematică

## DESCRIERE
- Întrebări create de Python pe baza cerințelor mele (adunare, scădere, înmulțire) ce sunt realizate în intervale prestabilite.
- Testul e compus din 6 întrebări, ce sunt împărțite pe niveluri de dificultate diferite, deci 2 întrebări pentru fiecare nivel.
Nivel 1: Numere mici (ex: 5-30), calcule simple.
Nivel 2: Numere până la 100.
Nivel 3: Numere mari (sute) sau înmulțiri mai complexe.
- Întrebările au o durată de 45 de secunde pentru a răspunde, iar la finalul fiecăruia primești verificarea dacă e corect sau greșit ce ai ales tu.
- Răspunsuri sub formă de grilă în intervale apropriate de răspunsul corect, există 4 variante din care să alegi.
- La final se contorizează scorul, ți se cere numele și vei fi pus într-un clasament ce este stocat într-un fișier de tip text.
